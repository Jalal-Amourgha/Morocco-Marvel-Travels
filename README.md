## Features

- **Home Page**: Discover featured accommodations and top destinations in Morocco.
- **About Page**: Learn more about our mission and the team behind Morocco Marvel Travels.
- **Contact Page**: Get in touch with us for any inquiries or support.
- **Book Page**: Browse and book hotels, houses, and apartments in Morocco.
- **Collection Page**: Save your favorite accommodations in local storage without registration.
- **Register Page**: Create an account to manage your bookings and collections.

## Technologies Used

- **Next.js**: React framework for server-side rendering and static site generation.
- **TypeScript**: Typed superset of JavaScript for better code quality and maintainability.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **MUI React**: React components library for faster and easier web development.
- **bcrypt.js**: Library for hashing passwords to enhance security.
- **MongoDB**: NoSQL database for storing user data and booking information.
